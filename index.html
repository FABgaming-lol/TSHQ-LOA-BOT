3. Warning System

Follow the 3-Strike Rule for minor offenses.

/warn [user] [reason]

Logs warning in DB and sends a DM to the user.

/check_warnings [user]
/unwarn [user]
/clear_warnings [user] Full Mods Only

Manager Alert

If a user hits 3 Warnings, Managers are pinged immediately for review.

7. Roblox Verification

Users must verify their identity to gain access to the HQ.

1. Click Verify

User initiates the process via the verification button.

2. Profile Code

Bot provides a unique code for the user's Roblox "About Me" section.

3. Auto Sync

Bot verifies the profile, renames the user, and grants access.
